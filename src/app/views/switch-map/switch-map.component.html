Switch map operator

El switch map es similar a map, sólo que esta vez no transforma el valor que retorna de un valor a otro, sino que tomar el valor de un observable y lo usa para generar otro observable. Esto se utiliza en Angular por ejemplo para obtener algún valor de la ruta que luego se requiere para utilizar otro observable:

Parámetros de la URL:

{{params$ | async | json}}

Contador que suma 1 cada "num" segundos:

{{routeCounter$ | async}}

Contador de referencia (no usa switch map) que cuenta cada segundo:

{{referenceCounter$ | async}}

Contador que suma 1 cada "num" + 2 segundos

{{routeCounterPlus2$ | async}}
